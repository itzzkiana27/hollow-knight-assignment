Run these commands from the HollowKnight project root (the folder containing build.gradle):

1. Extract this ZIP into the project root.
2. Test the patch:
   patch --dry-run -p1 < breakable_wall_custom_sfx_fix.patch
3. Apply the patch:
   patch -p1 < breakable_wall_custom_sfx_fix.patch

The sounds are installed at:
assets/audio/sfx/breakable_wall/breakable_wall_hit_1.wav
assets/audio/sfx/breakable_wall/breakable_wall_hit_2.wav
assets/audio/sfx/breakable_wall/breakable_wall_destroyed.wav
